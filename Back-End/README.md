# AI Agent System with LangGraph

A modular AI agent system built with Express, LangGraph.js, and Groq API.

## Features

- **Multi-agent architecture** with specialized agents (Researcher, Analyzer, Writer, Coordinator)
- **Workflow graphs** using LangGraph for complex task orchestration
- **WebSocket support** for real-time communication
- **Type-safe validation** with Zod
- **Environment-based configuration**
